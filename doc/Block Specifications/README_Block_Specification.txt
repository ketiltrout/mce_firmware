Copy "Block Specification.dot" into your word template directory:

C:\Documents and Settings\(your user name)\Application Data\Microsoft\Templates\

And you should be able to use it from the New Documents dialog box in Word.


The Visio stencil templates need to be copied to the following directory:

C:\Program Files\Microsoft Office\Visio 10\1033\Solutions\

Which will allow you to access them from the File -> Stencils menu item
